🚀 Freqtrade 配置管理器

🔗 API 连接设置

⚙️ 配置操作

📝 配置编辑器 (JSON)

✅ 可安全修改的配置（运行时可更改）

  • max_open_trades - 最大持仓数量
  • stake_amount - 每次交易金额
  • dry_run - 是否模拟交易
  • tradable_balance_ratio - 可交易资金比例
  • minimal_roi - 止盈配置
  • stoploss - 止损比例
  • trailing_stop - 追踪止损
  • pair_whitelist - 交易对白名单

⚠️ 谨慎修改的配置（建议重启Bot）

  • exchange - 交易所配置（key/secret等）
  • api_server - API服务器设置
  • pairlists - 交易对筛选规则
  • timeframe - K线周期
  • trading_mode - 交易模式（现货/合约）

⚠️ 使用说明

1. 修改配置后点击"保存配置"保存到文件
2. 点击"重载Bot配置"使更改生效（安全方式）
3. 如遇问题，点击"重启Bot"完全重启服务
